## Install
#### Native: (macOS x64 & Silicon, Linux x64, Windows Subsystem for Linux)
```
  curl -fsSL https://bun.sh/install | bash
```
#### Docker: (Linux x64)

```
docker pull jarredsumner/bun:edge
docker run --rm --init --ulimit memlock=-1:-1 jarredsumner/bun:edge
```
